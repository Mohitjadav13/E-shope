## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Mohitjadav13/my-ecommerce.git
```

2. Install dependencies:
```bash
cd my-ecommerce
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open http://localhost:5173 in your browser
